# 🌟 RoshniAI — Multi-Agent University Guide

> Illuminating Every Student's Path Worldwide

RoshniAI is a multi-agent AI system that helps students 
worldwide find the best universities and scholarships 
using 3 specialist AI agents working together.

## 🤖 Agent Architecture

- **Agent 1 — Profile Agent**: Collects student information
- **Agent 2 — University Finder**: Searches 30+ countries
- **Agent 3 — Scholarship Hunter**: Finds funding opportunities

## 🎯 5-Day AI Concepts Used

- ✅ Day 1: Multi-Agent System (3 specialist agents)
- ✅ Day 2: MCP Server + Agent Memory + A2A Communication
- ✅ Day 3: Agent Skills (SKILL.md)
- ✅ Day 4: Security Evaluation + Input/Output Validation
- ✅ Day 5: Web Deployment (Flask)

## 🌍 Countries Covered

Pakistan, USA, UK, Turkey, Malaysia, Germany, Japan, 
China, Korea, Singapore, Australia, Canada, India, 
Dubai, Qatar, Saudi Arabia, Ireland, Switzerland and more!

## 🚀 How To Run

1. Clone this repository
2. Install requirements:
pip install groq flask python-dotenv mcp

3. Create .env file:
GROQ_API_KEY=your_key_here

4. Run the web app:
python app.py

5. Open browser: http://localhost:5000
